# Go mockery
To demonstrate use of mockery package in unit testing.

## Tech stacks
Golang -version(Go 1.18)

External packages-
    
    Mux router
    Mockery
## Setup
To run the project

Run - make run
## mocking interface
To mock the interface we use this command . 

mockery --dir=user/model --name=Repo --output=user/model/mocks --outpkg=mocks
